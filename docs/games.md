# Available games

This trimmed build supports exactly three games.

Status | Game | Short name | Players | Deterministic | Perfect info | Description
------ | ---- | ---------- | ------- | ------------- | ------------ | -----------
Available | Carcassonne | `carcassonne` | 2 | No | Yes | Tile-placement game with meeples and terminal zero-sum win/loss returns.
Available | Othello | `othello` | 2 | Yes | Yes | Reversi/Othello on the standard board.
Available | Tic-Tac-Toe | `tic_tac_toe` | 2 | Yes | Yes | Players place tokens to form a three-in-a-row pattern.
